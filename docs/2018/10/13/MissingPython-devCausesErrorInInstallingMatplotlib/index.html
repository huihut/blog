python-dev 库缺失导致安装 matplotlib 出错

发表于 2018-10-13 | 分类于 CS

表现

安装 matplotlib 时出现以下错误

原因

无法找到 python 库，#include "Python.h" 出错

解决

  • 使用 apt (Ubuntu, Debian…) 安装

  • 使用 yum (CentOS, RHEL…) 安装

  • 使用 dnf (Fedora…) 安装

  • 使用 zypper (openSUSE…) 安装

  • 使用 apk (Alpine…) 安装
