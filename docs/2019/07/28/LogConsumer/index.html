一个基于 C# 的简单的线程安全日志模块

发表于 2019-07-28 | 分类于 CS

一个基于 C# 的简单的线程安全日志模块，它使用生产者 - 消费者模式，可以在 .NET Framework 和 .Net Core 中使用。

Github 地址：LogConsumer

使用

  1. 将 LogConsumer.cs 添加到你的项目中
  2. 将 LogConsumer.cs 中的 logFileName 修改为你的路径文件名
  3. 在需要输出日志的地方使用它

编译运行

  • 命令行
  • Visual Studio
    1. 打开 LogConsumer.sln
    2. 生成解决方案，运行测试
